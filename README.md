# Workshop Assets — AI for Fundraising

Resources from the workshop: **"Using AI to unlock immediate impact from messy data, fix conversion friction, and move to rapid testing."**

> **⬇️ Download everything:** Click the green **Code** button at the top of this page → **Download ZIP**. No GitHub account needed.

These tools cover three practical lenses you can apply to your organisation this week.

---

## 01 · Donor Export — Revenue Forensics

**What it is:** A prompt for any AI assistant (Claude, ChatGPT, Gemini) that analyses a raw donor CSV export and surfaces overlooked segments, lapse patterns, upgrade opportunities, and operational blind spots.

**What you need:**
- A donor export from your CRM (even a messy one)
- Access to any AI chat tool

**How to use it:**
1. Open `revenue-forensics-prompt.md`
2. Follow the instructions — paste the prompt into your AI tool and attach your CSV
3. Use `workshop-donor-export-sample.csv` to try it first if you don't have your own data to hand

---

## 02 · Campaign Attribution — GA4 Analysis

**What it is:** A guide for exporting session and traffic source data from GA4 and using AI to identify which campaigns and landing pages are converting — and which are not.

**What you need:**
- GA4 access (read-only is fine)
- Any AI chat tool

**How to use it:**
1. Read `ga4-analysis-guide.md` for the step-by-step export and analysis process
2. Use `ga4-sample-report.md` to see what a completed analysis looks like

---

## 03 · Donation Journey — Audit Rubric

**What it is:** A Node.js tool that uses a real browser to scan your donation journey, score it across 7–9 dimensions, and produce a prioritised report with estimated conversion impact for each finding.

**What you need:**
- A laptop with internet access
- ~5 minutes for one-time setup

**How to use it:**
→ Open `03-donation-journey/Audit Instructions - Start Here.md` and follow the steps.

---

## 14-Day Test Brief

**What it is:** A one-page hypothesis template to turn any finding from today into a structured test you can run this week.

**How to use it:**
1. Pick a finding from any of the three lenses
2. Fill in the hypothesis template: *"If we [change], then [audience] will [do X], measurable in 14 days"*
3. Run it. Read the result on day 14. Ask the next question.

→ Open `14-day-test-brief.md`
